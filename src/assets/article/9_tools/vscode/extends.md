# 云端保存 Extends

- Settings Sync

  在 Mac 中如果使用快捷键，上传/下载 仍然无法工作，则可以使用 `shift + command + p`，然后输入 `sync`，找到 `Sync: Dowload Settings` 和 `Sync: Update/Upload Settings`，最后选择下载或上传即可。
